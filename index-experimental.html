How the Rotation Planner works

Build a Persona 5: The Phantom X boss rotation, then export it as a shareable image or a JSON file. Everything you type is saved automatically in your browser.

Toolbar

  • Guide — opens this help.
  • Load Examples — load a ready-made MLD or DOD rotation to see how a finished plan looks. This replaces your current work, so export first if you want to keep it.
  • Import — load a rotation:
    • Import JSON — from a .json file (e.g. one a teammate shared).
    • Import Code — from a share code, full link or short link pasted into the box.
    • Import Text — paste a rotation written in plain text (like a post from Discord) and the planner fills in what it recognizes: boss, mode and patch from the title, the team with awareness/cards, personas, the dagger, and every turn. Anything left empty (or anything it could not read) is highlighted in yellow so you can see at a glance what still needs your attention, and a summary lists whatever could not be imported. If a rotation is already open, it asks before replacing it. Tick Keep the pasted text in a movable window to keep the original text in a small draggable panel afterwards, so you can refer to it while you tidy up the turns.
  • Export — save or share your rotation:
    • JSON — a data file you can re-import later or share.
    • PNG & Images — an image with character portraits, persona art and the Wonder's dagger.
    • PNG (no Images) — the same layout as text only.
    • PNG (Rotation only) — just the turns, without the team section.
    • Text — a plain-text version of the whole rotation (a title line with patch / mode / name, then credits, boss & personas, team, notes and all turns) in a box with a Copy button, ready to paste into Discord. In the turns, Wonder actions are listed under their persona; actions without a persona (e.g. Attack / Guard / Gun) use the Wonder name. Action labels use the short codes (HL, MSQ, S1…).
    • Share Editor Link — a link that loads the whole rotation into the editor when opened.
    • Share View Link — a link that opens a read-only view which looks like the image export (no editing controls, mobile-friendly, with a PNG download button), ideal for sharing a finished rotation.
    • Copy Code — a short text code that holds the whole rotation.
  • Settings — local display options (saved only on this device): choose a Light, Dark or System theme (the image export always stays dark), show or hide the Character, Team and Rotation notes fields, put the persona images on the left, choose whether the rotation preview starts expanded, switch the rotation layout between vertical and horizontal, and export or import your settings as a JSON file. These never affect your rotation, share links or exports.
  • Reset — clear all fields and start a blank plan.

Setup (top bar)

Fill in the header: Rotation Name, Mode (MLD / DOD / NOD / SOS), Boss, Patch, Expected Score and Credits. Empty fields are simply left out of the export, but it is recommended to fill in all of them.

Team

There are four team slots (1st–4th) plus the Elucidator. For each unit pick a Character — the field works like the Boss and Persona fields (click to see the full list, or type to filter it; only names from the list are accepted). You can also type an in-game name or nickname (e.g. Manaka or Navi) and the matching codename is offered — the codename is what gets saved. A small × appears inside the field (and in the Boss field) once it has a value; tapping it clears the field and opens the list right away — handy on mobile. Then set:

  • Awareness — A0 to A6.
  • Reforge — RX to R6.
  • Revelation Cards — choose the Space card first, then the matching Sun / Moon / Star / Sky cards are shown. For most characters their usual set is filled in automatically when you pick them; you can still change it. A text import that names a set always takes precedence over the default.
  • Special fields appear automatically: Wonder gets a Dagger (its artwork appears in its own box next to the Personas), Twins a Role (which shows the matching S1 / S2 dual-element symbols below it, and in the image export too), Fuuka a Backup.
  • Each character has its own Notes field at the bottom of the card; it also appears at the end of the character's box in the image export.

You can reorder team slots 1–4 by dragging them. The Whale button in the team header sets every Awareness to A6 and every Reforge to R6 at once.

Below the team you set up to three Personas (with up to two teachable skills each) and a Team Notes box, where you can add information such as the required Pierce or Crit. The Persona field works like the Boss field — click it to see the full list, or type to filter it (only names from the list are accepted). Each persona shows its Signature skill on a fixed line above the two teachable-skill fields — it is set automatically by the persona and can't be edited. The two teachable-skill fields are combo boxes: pick from the skill list, or type freely (free text is allowed, and abbreviations like “WT” or “Taru” filter the list). In a text import, writing Sig as a persona's skill is understood as that persona's signature skill, and common skill abbreviations are expanded to their full names.

Rotation

The rotation is a row of turns; each turn has a name (which you can edit), an optional turn note field below the name, and a list of actions. The turn note is shown in the export next to the turn name — in the vertical layout right-aligned in the header, in the horizontal layout under the turn name.

Start here: press Auto Fill first. It fills every empty turn column with your team characters 1–4 in order, so afterwards you only have to fill in the actions. It is recommended to do this before adding actions yourself. Afterwards Auto Fill never overwrites a column that already has a character, so if you add more turns later you can press Auto Fill again to fill them. Clear removes all actions from every turn if you want to start over.

If you swap a team character or a Persona while actions still reference the old one, those actions are highlighted in red (in the editor and the preview) and the Export button is disabled until you fix them — this prevents exporting a broken rotation. A banner briefly appears to point it out. As soon as every action again points to a current character/Persona, export is re-enabled automatically.

Export is also disabled while the Setup is incomplete. Every setup field is required except Expected Score, which is optional. Patch must be a number like 4.3, and Expected Score accepts plain numbers, grouped numbers like 1,000,000 or 1.000.000, or magnitudes such as 100M, 1.3 B, 945 Mil, 2Bil or 59T. A decimal comma like 1,3 is converted to a period automatically, and a lower-case T/M/B is capitalised for you. An invalid field turns red and a tooltip explains the problem with an example.

When there is a discrepancy, an Auto Correct button appears next to Auto Fill. It opens a dialog with a checklist of every change it can make, each one toggleable, so you stay in control. It can: replace a missing character in the actions with the one that took its team slot; fix a Wonder Persona that is no longer in a slot (swapping it for a free slot persona, or dropping it into an empty slot); and resolve turn order. For the order it offers a choice — reorder the team slots to match the turns, reorder the actions inside each turn (keep both, fix the order), or delete the too-early or the too-late action in each conflicting turn. When characters act in conflicting orders across different turns (so no single team order can satisfy them all), it says so and recommends reordering the actions instead. After applying, an Undo appears so you can revert everything in one click. Anything it genuinely can't resolve (e.g. a missing character with no free slot) is listed for you to fix by hand.

A turn is also tinted red when the characters in it don't act in team order: the planner reads the first Skill/Attack/Gun/Guard of each character (ignoring Highlights/Theurgy, Assists and the Elucidator) and compares it to the order of your team slots. This is enforced the same way — the banner reports it and Export stays disabled until the turn matches the team order again. Reordering your team (or editing a turn) re-checks every turn instantly, so it's easy to spot a sequence that slipped.

For every action:

  • Pick the acting character from the dropdown.
  • Optionally tap an action button. Normal characters have S1 S2 S3 HL plus Attack Gun Guard; SEES members show TG (Theurgy) instead of HL and Assist instead of Gun; the Elucidators only have S1 S2 S3. A few characters (Messa, Violet, Rin·F, Turbo, Twins, Ange, Akihiko) also have a separate button next to the character dropdown that exports to its own wording — e.g. Violet's MSQ → “Masquerade”, Ange's DC → “Da Capo”, Akihiko's Alt → “Punch”. Tap an active button again to clear it.
  • Wonder is special: choose one of your three personas from the dropdown and optionally toggle the HL button next to it. A second row of buttons (Attack / Gun / Guard) covers Wonder's basic actions. Once a persona is selected, a skill dropdown appears with that persona's three skills (its signature plus the two teachable skills) so you can pick the exact skill quickly. In a text import, a bare Wonder HL automatically adopts the persona Wonder used most recently.
  • The text field is always available as the last line for free notes, e.g. the target of the action or other information.

In the export, a button and text are joined with a space — e.g. S2 + Violet becomes “S2 Violet”, and a Wonder action becomes e.g. “Koumokuten HL Haru”. You can also use only the text field without any button.

Each turn header has a grip (drag to reorder the column), a duplicate button (copies the turn with all its content to the right) and a × button (remove). You can also drag actions into a new order. Turns stay numbered left to right (1, 2, 3, …); in DOD mode the last two turns are breaks (Break 1, Break 2), always pinned at the end and highlighted in red in the export. A separate Rotation Notes box sits above the turns.

Rotation Preview

The Rotation Preview toggle below the turns shows a live preview of how the rotation will look in the image export. It updates as you type and is never part of the exported file.

Saving & sharing

Your work is saved automatically in this browser (autosave). For a permanent backup, use Export → JSON and re-import the file later. For a picture to post, use one of the PNG exports.

To send a rotation to a teammate without a file, use Export → Share Editor Link or Copy Code. Both open a box showing the link/code with a Copy button. The two share-link boxes (Share Editor Link and Share View Link) automatically turn the link into a short one (via a small Cloudflare service) the moment they open. The same rotation always reuses the same short link — a new one is only created when the rotation actually changes. They also show an editable Discord title line, pre-filled with the rotation name, mode, patch and credits. Adjust the title however you like, then press the Discord button to copy a masked link (your title as a single clickable label, with the link hidden behind it) ready to paste into Discord. Opening a Share Editor Link loads the rotation automatically; a Code is loaded via Import → Import Code (which also accepts a full link or a short link). The whole rotation is packed into the link/code itself — nothing is uploaded to a server.
